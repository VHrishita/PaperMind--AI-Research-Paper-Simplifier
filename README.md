# 📄 PaperMind - AI Research Paper Simplifier

PaperMind is an AI-powered research paper analysis platform that helps students, researchers, and professionals **upload, understand, simplify, compare, and interact with research papers effortlessly**.

Built with a clean frontend, Flask backend, Firebase authentication, and advanced NLP workflows.

---

## ✨ Features

- 🔐 Firebase Login / Signup Authentication
- 📄 Upload and preview PDF research papers
- 💬 Chat with research papers (Q&A)
- 🧠 AI-powered summarization
- ✨ Simplify complex academic language
- 🔍 Keyword extraction
- 🧩 Automatic section detection
- ⚖️ Compare multiple papers
- 📊 Topic cluster visualization (PCA)
- 🌐 Concept explorer using Word2Vec
- 📥 Export analysis reports as PDF
- ☁️ Cloud-ready frontend + backend deployment
